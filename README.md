## Linguistic Analysis of Tweets
Using Data Mining to Study the Usage of English on Twitter

 ABSTRACT

The evolution of everyday language has significantly transformed the way people communicate in digital environments. With the widespread adoption of social media platforms, there has been a notable increase in the use of abbreviations, shorthand expressions, emojis, and hybrid words. This shift has led to a deviation from traditional English vocabulary and grammatical structures.

This study focuses on analyzing linguistic patterns in user-generated content collected from Twitter, a widely used social media platform where users express opinions and emotions in real time. The dataset consists of randomly sampled English tweets, making it suitable for unbiased linguistic analysis.

Natural Language Processing (NLP) techniques are applied to preprocess and analyze the textual data. The raw tweets are cleaned by removing noise such as URLs, hashtags, mentions, numbers, and special characters. Further processing includes tokenization and stopword removal to extract meaningful linguistic units.

Each word in the dataset is categorized into two groups: Proper English Words, which belong to standard vocabulary, and Improper English Words, which include slang, abbreviations, and non-standard lexical forms. This classification helps in understanding the extent of deviation from formal English usage in social media communication.

The study provides insights into modern linguistic trends and highlights the impact of digital communication on language evolution. It also demonstrates how data mining and NLP techniques can be used to analyze large-scale textual data effectively.
